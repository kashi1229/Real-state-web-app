## Tech Stack & Design Direction

### Dependencies (already installed)
- React 19 + TypeScript
- Tailwind CSS 4 (with @tailwindcss/vite)
- Vite 8
- React Router DOM v7
- GSAP 3.15 (ScrollTrigger, ScrollToPlugin)
- Framer Motion 12
- Lucide React icons 1.23

### Additional Libraries to Install
- `lenis` — Smooth scrolling with native-like feel
- `react-hot-toast` or `sonner` — Toast notifications
- `recharts` — Charts for admin analytics

### Design System
- **Design tone**: Clean, trustworthy, slightly upscale — not flashy. North American real estate buyers associate polish with credibility.
- **Color palette**: Deep forest green (#1a3c34) + warm sand (#e8d5b7) + charcoal (#2d2d2d) + cream (#faf6f0) + brass gold accent (#c9a96e)
- **Typography**: 
  - Headlines: Playfair Display (serif — trust/established feel) via Google Fonts
  - Body/UI: Inter (clean sans-serif) via Google Fonts
- **Animation**: GSAP ScrollTrigger for entrance animations, parallax effects, and counter animations. Framer Motion for UI micro-interactions. Lenis for smooth scrolling.
- **Iconography**: Lucide React icons throughout
- **One signature element**: Interactive live search/filter with smooth GSAP transitions
